# Web Calculator (Django Based)

## 📖 Description
Web Calculator is a web-based calculator developed using the Django framework.  
It performs basic arithmetic operations (addition, subtraction, multiplication, and division) directly in the browser using Python views and templates.  

This project demonstrates Django routing, template rendering, and frontend styling with HTML and CSS.

---

## 🚀 Features
- Perform addition (+), subtraction (-), multiplication (×), and division (÷)
- Clean and responsive user interface
- Input handled via Django views and templates
- No database required

---

## 🛠 Technologies Used
- Python 3
- Django Framework
- HTML5
- CSS3
- Git & GitHub

## ▶ How to Run the Project

1. Clone the repository:
```bash
git clone https://github.com/your-username/web-calculator.git
```

2. Move into the project folder:
```bash
cd web-calculator
```

3. Create a virtual environment:
```bash
python -m venv venv
```

4. Activate virtual environment:

**Windows:**
```bash
venv\Scripts\activate
```

5. Install Django (if not already installed):
```bash
pip install django
```

6. Run the development server:
```bash
python manage.py runserver
```

7. Open the project in your browser:
```
http://127.0.0.1:8000/
```
